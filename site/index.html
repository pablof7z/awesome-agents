How it works

A profile is a whole agent, not a prompt snippet.

Every profile ships with an open definition and a record of what it has actually done. You audit first, install second.

01

Read the definition

Open the exact prompt, model, and tool boundaries before anything touches your machine. No black boxes.

02

Check the shipped work

Each profile publishes real, user-approved samples. Judge it on output, not on a marketing blurb.

03

Install & run

One command adapts the profile to your harness and drops it in place. Run it with the CLI you already use.
