Source format

The repo-neutral layout an agent-profile source uses.

Directory layout

An agent-profile source is a folder of agents/<slug>/ definitions:

agents/
  triage-agent/
    agent.yaml
  ops-agent/
    agent.agf.yaml
    skills/
      gh-pages-publisher/
        SKILL.md
    scripts/
      heartbeat.sh
    references/
      runbook.md

YAML profile files are preferred. The loader also accepts Markdown files with YAML frontmatter for compatibility with .agent.md-style profiles.

Owned scripts and references

Agent-owned scripts/ and references/ are installed into ~/.agents/homes/<slug>/scripts and ~/.agents/homes/<slug>/references, never into a harness configuration directory.
