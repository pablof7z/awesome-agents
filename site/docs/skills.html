Attach immediately relevant skills to a profile.

Declaring skills

A profile can declare the skills it needs:

skills:
  - gh-pages-publisher
  - pablof7z/tenex-edge basic-skill

Resolution

Bare skill names are resolved from the profile directory, then the source checkout, then ~/.agents/skills. Source-qualified entries use the same source-plus-skill selector shape as npx skills add <source> --skill <skill>.

Declared skills are copied into ~/.agents/homes/<slug>/skills/<skill> and listed with complete paths in the installed agent prompt.
