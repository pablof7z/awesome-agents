Getting started

Installation

Install a profile from npm, or from a local checkout during development.

From npm

Run it straight from npm with npx. Point it at a source and pick an agent:

$ npx awesome-agents add owner/repo --agent triage-agent
$ npx awesome-agents add owner/repo --agent triage-agent --harness tenex-edge

Without --harness, the CLI detects the harness CLIs on your PATH and installs for each. If none are detected, pass --harness explicitly.

From a local checkout

Working on a source repo? Run the CLI from the checkout and preview writes with --dry-run:

$ npm install
$ npm test
$ node ./bin/awesome-agents.js add ./test/fixtures/profile-source --agent triage-agent --dry-run
