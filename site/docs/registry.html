Registry & safety

How the CLI tracks installs and protects your files.

The registry

The CLI keeps its own registry at .awesome-agents/installed.json for project installs, or ~/.awesome-agents/installed.json for global installs. list, remove, and update read from it.

Overwrite safety

Generated files are marked Generated by awesome-agents. The CLI refuses to overwrite or delete any file that does not contain that marker unless you pass --force, so it never clobbers something it did not write.
