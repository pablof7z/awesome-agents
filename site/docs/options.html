Usage

Install options

Flags that control what gets installed, where, and how.

Selecting a profile

  • --agent <slug> selects an agent profile, e.g. --agent triage-agent.
  • --profile <slug> or --skill <slug> are explicit aliases. --skill is command-shape compatibility only; it does not mean the artifact is a skill.
  • Omit a selector in an interactive terminal to choose from a checkbox list; every profile is selected by default.

Selecting harnesses

  • --harness codex|claude-code|opencode|tenex-edge|* targets specific harnesses. Without it, the CLI detects harness CLIs on PATH.
  • --all installs every profile to every supported harness.
  • --yes accepts the detected profile and harness selections without opening selectors.

Scope

  • --project installs at project level where supported; --global installs at user level.
  • Codex and tenex-edge always install globally.

Preview and output

  • --dry-run previews writes without touching disk.
  • --list inspects available source profiles without installing.
  • --json for machine-readable output; NO_COLOR=1 disables color.
