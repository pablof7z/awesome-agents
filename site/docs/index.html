Getting started

Introduction

What awesome-agents installs, and why it lets you audit before you run.

New here? Jump to Installation to run your first install, or browse the directory of published profiles.

What you install

awesome-agents is an npx-compatible installer for reusable agent profiles. It mirrors the useful parts of npx skills, but the unit is a whole operational agent profile, not a skill: its prompt, model, and tool boundaries.

A profile is read from a YAML or Markdown file under agents/<slug>/, adapted for the harness you target, and written into the right place for Claude Code, Codex, OpenCode, or tenex-edge. Sources are GitHub repos, Git URLs, or local checkouts.

Read before you run

Every profile ships with an open definition and a record of what it has actually done. You read the exact prompt and tool boundaries, check the published samples, and only then install. Nothing is a black box.

Command shape

The commands follow npx skills: add to install, use to render without installing, plus list, remove, update, and init. Everything is scriptable; pass --json for machine-readable output.
